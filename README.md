# 配管勾配計算 Pro

現場用の配管勾配計算アプリ（汚水・雨水）｜設備AIシリーズ

## ステータス

**配管勾配計算Pro 正式版 完成**

- 完成日：2026年7月22日
- 現在のバージョン：1.0.4
- 公開URL：https://chihikan.github.io/pipe-slope-calculator-pro/

## 実装済みの主要機能

- 現場名・工事名・作業日・メモ等の現場情報管理
- 汚水・雨水それぞれの本管・枝管の複数ライン管理（追加・切替）
- 配管勾配の自動計算、地点ごとのOK／切る／盛るの実測判定
- レベル計（器械）の設置記録・盛替え管理
- 設計GL照合記録
- 実測値の入力に応じた連鎖再計算
- 元に戻す／やり直す（Undo／Redo）
- 現場データの自動保存・保存・読込（localStorage、ファイル入出力）
- Excel（.xlsx）出力、CSV出力、印刷／PDF出力
- LINE・メール等への共有
- PWA対応（オフライン利用、ホーム画面へのインストール、Service Workerによるキャッシュ管理）
- スマートフォン・タブレットでの操作を想定したUI

## ファイル構成

- `index.html` — アプリ本体
- `manifest.json` — PWAマニフェスト
- `sw.js` — Service Worker（オフラインキャッシュ）

## 今後の開発方針

正式版（本リポジトリの現状）を基準とする。現場での使用を通じて見つかった改善点・不具合は、正式版を壊さないようバックアップとバージョン管理を行った上で、個別に対応する。

詳細な経緯は [開発日誌.md](開発日誌.md) を参照。
